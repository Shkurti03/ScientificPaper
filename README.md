# Survey Provenance - PROV Diagram

## Description
A PROV diagram representing the provenance of a student financial 
support survey, built as part of a practical exercise on representing 
provenance using the PROV model.

## Task 1 - PROV Diagram
The diagram models the following scenario:
- Laura designs a survey about student financial support
- Jack and Jill conduct the survey and collect data (1st round)
- A year later, Laura revises the survey
- Peter and Paula conduct the revised survey (2nd round)
- Zack compiles all results, analyzes them with a statistics package,
  and publishes a paper with Laura and other co-authors

### PROV Elements represented:
- Entities, Activities, Agents
- wasGeneratedBy, used, wasAssociatedWith
- wasDerivedFrom, wasRevisionOf, hadPlan

## How it was created
The diagram was created using HTML, CSS and JavaScript on CodePen,
representing the PROV-DM model visually.
